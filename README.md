# Pakkausharjoitus

Harjoitustyö Tietorakenteet ja algritmit -harjoitustyökurssille. Työssä toteutetaan tiedoston pakkaaminen ja purkaminen sekä Huffman- että Lempel Ziv-algoritmilla, ja vertaillaan niitä.
